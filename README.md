# Embedded Linux Resource Monitor

## Overview

Embedded Linux Resource Monitor is a Linux system monitoring tool developed in C. It collects and displays system information using the Linux `/proc` filesystem.

## Features

* CPU Information
* Memory Information
* Disk Information
* System Uptime
* Load Average Monitoring
* Process Monitoring

## Project Structure

```text
include/
src/
Makefile
```

## Build

```bash
make
```

## Run

```bash
./monitor
```
